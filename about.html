JavaScript is a scripting or programming language that allows you to implement complex features on web pages
Variables are used to store information to be referenced and manipulated in a computer program
Functions are "self contained" modules of code that accomplish a specific task. Functions usually "take in" data, process it, and "return" a result.
An argument is a way for you to provide more information to a function.
In computer programming, a parameter or a formal argument is a special kind of variable used in a subroutine to refer to one of the pieces of data provided as input to the subroutine.
Loops are used in JavaScript to perform repeated tasks based on a condition. Conditions typically return true or false when analysed. A loop will continue running until the defined condition returns false.
Object Oriented programming (OOP) is a programming paradigm that relies on the concept of classes and objects. All of the relevant data and functions for the thing that the variable name represents are kept together in a "capsule", better known as an object.
